Setup Instructions

VoiceForge uses Coqui XTTS v2 — a free, open-source, locally-running TTS engine.

1. Install dependencies
pip install TTS flask
2. Install FFmpeg (for audio conversion)
# macOS
brew install ffmpeg

# Ubuntu/Debian
sudo apt install ffmpeg

# Windows: download from ffmpeg.org
3. Run the server
python app.py
4. First run: model download (~2GB)
# XTTS v2 model will auto-download on first run.
# Subsequent runs load from cache instantly.

Everything runs locally. No API keys. No internet needed after setup.
